# 📰 아침 뉴스 스크랩

전력·전기직 취업 준비용 **데일리 뉴스 브리핑**을 보기 좋은 웹 리더로 모아 보는 정적 사이트.

- 매일 아침 생성되는 `뉴스스크랩_YYYY-MM-DD.md` 파일들을
- `generate.py`가 하나의 자체 완결형 `index.html`로 빌드하고
- GitHub에 push되면 Vercel이 자동 배포.

## 구성

| 파일 | 역할 |
|---|---|
| `뉴스스크랩_*.md` | 매일 생성되는 원본 스크랩 (소스이자, 사이트가 직접 불러오는 데이터) |
| `generate.py` | md들을 스캔해 **가벼운 목록(manifest)** 만 담은 `index.html` 빌드 |
| `index.html` | 홈/목록은 즉시 표시, **본문은 날짜를 열 때 해당 md만 fetch(지연 로딩)** |

> 지연 로딩 덕분에 날짜가 수년치 쌓여도 첫 로딩이 가볍습니다.
> 본문 열람에는 (md를 fetch하므로) 웹 서버가 필요해요 — Vercel에서는 그대로 동작하고,
> 로컬에서 볼 땐 `python -m http.server` 로 띄우면 됩니다.

## 매일 업데이트 (체이닝 방식)

새 md가 생성된 직후 아래 한 묶음을 실행하면 빌드 + 배포가 한 번에 됩니다.

```bash
python generate.py
git add -A
git commit -m "뉴스 $(date +%Y-%m-%d)"
git push
```

→ push되면 Vercel이 자동으로 감지해 새 브리핑을 배포합니다.

## 기능

- 왼쪽 날짜 목록 / 날짜·**본문 전체 검색**(일치 부분 형광펜 강조)
- 사이드바 접기/펼치기
- 뉴스 카드 + 출처 링크 칩, 태극기 인라인 SVG, Pretendard 글꼴, 다크모드 자동
